What is this site about?

This site presrnts a tool for automatically generating marking for electrical equipment. Electrical products should have all of its components marked with stickers. The tool automates generation of marking. You only have to provide it with enumeration of electrical components` designation.

How to use it?

Go to dashboard and drop the input in. The input may be a .txt, .csv, .json, .yaml or .pdf file.

Here are sample inputs that demonstrate input structure:

    .txt example .csv example .json example .yaml example

Tool also accepts .pdf as an input, but it must be structured in a standartized way.
